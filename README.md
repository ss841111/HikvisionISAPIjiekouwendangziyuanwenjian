# Hikvision ISAPI 接口文档资源文件

## 描述

本资源文件提供了海康威视（Hikvision）ISAPI接口的详细文档，包含所有接口的请求头格式和body内容的XML格式。文档涵盖了以下接口：

- /ISAPI/AUXInfo
- /ISAPI/ContentMgmt
- /ISAPI/Event
- /ISAPI/Image
- /ISAPI/SDT
- /ISAPI/PTZCtrl
- /ISAPI/Security
- /ISAPI/Smart
- /ISAPI/Streaming
- /ISAPI/System
- /ISAPI/Thermal

通过本资源文件，开发者可以快速了解和集成海康威视设备的ISAPI接口，实现设备管理、事件处理、图像处理等功能。
